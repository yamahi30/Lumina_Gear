# note（無料記事・アフィリエイト）文体・構成ガイドライン
---

## 1. 無料記事の役割と基本方針

**役割**: 圧倒的な価値提供（GIVE）により、「この人は本物だ」という信頼を獲得し、有料記事やSNSフォローにつなげる「信頼構築とファン化の入口」。

**基本方針**: あなたのガイドラインを全面的に採用。読者の悩みに寄り添い、体験ベースで共感を生むことを最優先する。

**参考モデル**: chimacoro（GIVE戦略）, ayamiroom（体験談レビュー）

---

## 2. 文体：信頼できる友達のブログ

### 2.1 基本トーン

- **丁寧さ**: X/Threadsより丁寧。「です・ます」調で、方言は使わない。
- **一人称**: 私
- **言葉選び**: 「構造的な言語化」を使いつつ、読者に寄り添う丁寧な言葉を選ぶ。
- **感情表現**: 喜び、悲しみ、葛藤など、自身の体験に基づく感情を素直に表現する。

### 2.2 絵文字・装飾

- **絵文字**: 見出しや段落の締めに効果的に使い、読みやすさを演出する。Threadsの絵文字パレットを基本とする。
- **装飾**: **太字**や「」を効果的に使い、重要な部分を強調する。

## 3. 構成テンプレート

### 3.2 B. アフィリエイト付き記事

**目的**: 自然な紹介で収益化。「売る」のではなく「共有する」スタンス。

```
【タイトル】
（例）【家事嫌いな私へ】もっと早く買えばよかった…〇〇（商品名）で人生変わった話

【導入】
正直に言います。私は、〇〇が大嫌いでした。
毎日毎日、終わりのないタスクに追われて…。
そんな私が、藁にもすがる思いで手にしたのが、この〇〇です。

【レビュー：正直な感想】
■ 良かったところ
・〇〇
・〇〇

■ ちょっとイマイチなところ（正直に）
・〇〇
・〇〇

【こんな人におすすめ】
もしあなたが、
・〇〇で悩んでいる
・△△な時間がほしい
と思っているなら、きっとこの商品はあなたの救世主になります。

【まとめ】
もちろん、これが万能な解決策だとは言いません。
でも、私の人生が劇的に楽になったのは事実です。

もし興味があれば、こちらから詳細を見てみてくださいね。
[アフィリエイトリンク]
```

## 4. AI生成時チェックリスト

あなたのガイドラインにある**種類別の詳細なチェックリスト**をそのまま活用します。

### 4.1 共通チェックリスト

- [ ] 同じ語尾が2回以上連続していないか？
- [ ] 「〜しましょう」「いかがでしたか」が含まれていないか？
- [ ] 冒頭3行で読者の悩み・共感に触れているか？
- [ ] 見出しに絵文字パレット内の絵文字が使われているか？
- [ ] 短文×改行のリズムが保たれているか（1文3行以内）？

### 4.3 無料記事（アフィあり）チェックリスト

- [ ] アフィリエイトリンクが記事内1〜2箇所に収まっているか？
- [ ] 体験ベースで自然に紹介されているか（売り込み感がないか）？
- [ ] 「今だけ」「絶対」等の煽り表現がないか？
