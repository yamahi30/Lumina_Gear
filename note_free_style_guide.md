# note（無料記事・アフィリエイト）文体・構成ガイドライン
---

## 1. 無料記事の役割と基本方針

**役割**: 圧倒的な価値提供（GIVE）により、「この人は本物だ」という信頼を獲得し、有料記事やSNSフォローにつなげる「信頼構築とファン化の入口」。

**基本方針**: あなたのガイドラインを全面的に採用。読者の悩みに寄り添い、体験ベースで共感を生むことを最優先する。

**参考モデル**: chimacoro（GIVE戦略）, ayamiroom（体験談レビュー）

---

## 2. 文体：信頼できる友達のブログ

### 2.1 基本トーン

- **丁寧さ**: X/Threadsより丁寧。「です・ます」調で、方言は使わない。
- **一人称**: 私
- **言葉選び**: 「構造的な言語化」を使いつつ、読者に寄り添う丁寧な言葉を選ぶ。
- **感情表現**: 喜び、悲しみ、葛藤など、自身の体験に基づく感情を素直に表現する。

### 2.2 絵文字・装飾

- **絵文字**: 見出しや段落の締めに効果的に使い、読みやすさを演出する。Threadsの絵文字パレットを基本とする。
- **装飾**: **太字**や「」を効果的に使い、重要な部分を強調する。

## 3. 構成テンプレート

### 3.1 A. 無料記事（アフィリエイトなし）

**目的**: 信頼構築・ファン化。フックを重視し、体験ベースで共感を生む。

```
【タイトル】
（例）【完全無料】家事と育児に追われるあなたのための「家庭DX」最初の一歩

【導入】
「毎日やることが多すぎて、自分の時間なんて全くない…」
そんな風に感じていませんか？
この記事を読み終える頃には、その悩みを解決し、
1日30分の心の余裕を生み出す具体的な方法が分かります。

【本文】
まず結論から。大切なのは、完璧を目指さず「3つのこと」だけを試してみることです。

1. ○○（図解や画像を使いながら丁寧に解説）
2. ○○
3. ○○

【まとめ】
今日からできることは、この3つです。
完璧じゃなくて大丈夫。一つずつ、あなたのペースで試してみてくださいね。

【CTA】
もしこの記事が「役に立った」と感じたら、
ぜひXやThreadsもフォローしていただけると嬉しいです。
[SNSへのリンク]

もっと詳しく知りたい方は、こちらの有料noteで全てのノウハウを解説しています。
[有料noteへのリンク]
```

## 4. AI生成時チェックリスト

あなたのガイドラインにある**種類別の詳細なチェックリスト**をそのまま活用します。

### 4.1 共通チェックリスト

- [ ] 同じ語尾が2回以上連続していないか？
- [ ] 「〜しましょう」「いかがでしたか」が含まれていないか？
- [ ] 冒頭3行で読者の悩み・共感に触れているか？
- [ ] 見出しに絵文字パレット内の絵文字が使われているか？
- [ ] 短文×改行のリズムが保たれているか（1文3行以内）？

### 4.2 無料記事（アフィなし）チェックリスト

- [ ] フックが効いているか（最初の3行で引き込めるか）？
- [ ] 有料noteやメンバーシップへのさりげない導線があるか？
