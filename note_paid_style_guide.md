# note（低価格・高価格）文体・構成ガイドライン
---

## 1. 有料記事の役割と基本方針

**役割**: あなたの知識と経験を収益に変える「収益化の柱」。読者に価格以上の価値を提供し、熱量の高いファンへと育成する。

**基本方針**: あなたのガイドラインを全面的に採用。無料パートで「買うしかない」と思わせ、有料パートで120%の価値を提供する。

**参考モデル**: fukuda_CEO（権威性＋網羅性）, chimacoro（GIVE戦略）

---

## 2. 文体：信頼できる専門家

### 2.1 基本トーン

- **丁寧さ**: 「です・ます」調で、専門家としての信頼感と、読者に寄り添う丁寧さを両立させる。
- **一人称**: 私
- **言葉選び**: 専門用語も使いつつ、必ず分かりやすい言葉で補足説明を入れる。

### 2.2 絵文字・装飾

- **絵文字**: 見出しや重要なポイントに効果的に使い、視覚的な分かりやすさを助ける。
- **装飾**: **太字**、「」、『』、箇条書き、表などを駆使し、網羅的で読みやすい構成を心がける。

## 4. AI生成時チェックリスト

あなたのガイドラインにある**種類別の詳細なチェックリスト**をそのまま活用します。

### 4.1 低価格NOTEチェックリスト

- [ ] 無料パートでツールの結果・魅力が見せられているか？
- [ ] 無料パートで使い方を教えすぎていないか？
- [ ] 有料パートのセットアップ手順が初心者でもわかるか？
- [ ] ツールのダウンロードリンクが含まれているか？
- [ ] 購入者への感謝と「質問はコメントで」の一言があるか？

### 4.2 高価格NOTEチェックリスト

- [ ] 無料パートだけでも「価値がありそう」「ワクワクする」と感じるか？
- [ ] 無料パートで情報を抽象化しつつ、核心は有料パートに残しているか？
- [ ] 無料パートですべてを教えすぎていないか？
- [ ] 有料パートで具体的な数字・手順・テンプレートが全公開されているか？
- [ ] 価値提供120%と感じるか（「この値段で安い」と思えるか）？
- [ ] ツールが複数添付されているか？
- [ ] 各ツールの使い方が説明されているか？
- [ ] 購入者への感謝と「質問はコメントで」「アップデート追記」があるか？
