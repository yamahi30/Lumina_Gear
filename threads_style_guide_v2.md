## 1. 文体の基本方針

**一言で表すと**: 「HSPの友達が、今日あったことをカフェで話してくれてる感じ」

- Xより深く、じっくり語れる場所
- 「共感させる日記」がベース
- 短文×改行のリズムはXと共通。ただし段落が長めでOK
- 体験や感情を丁寧に描写して、読んでる人に自然と寄り添う
- 押し付けない、説教しない、ただ「一緒にいる」空気感

### Xとの違い

| | X | Threads |
|--|---|---------|
| 文字数 | 140字 | 500字 |
| スタイル | つぶやき・ワンフレーズ | 日記・ストーリー |
| 深さ | 気づきを一言で | 体験を丁寧に描写 |
| 構成 | フック→体験→締め（コンパクト） | 出来事→感情→気づき→寄り添い（じっくり） |
| 改行 | 1〜2行で改行 | 2〜4行のまとまりOK |
| 読後感 | 「わかる！」 | 「わかる…。なんか救われた」 |

## 2. 語尾ルール

### 2.1 メインで使う語尾

| 語尾 | 種類 | 使用頻度 | 例 |
|------|------|---------|-----|
| 〜んですよね | 標準 | ★★★ 主力 | 「気持ちの余裕がぜんぜん違うんですよね」 |
| 〜なんです | 標準 | ★★★ 主力 | 「ほんとにしんどかったんです」 |
| 〜しちゃって | 標準 | ★★☆ よく使う | 「自分でもびっくりしちゃって」 |
| 〜んですけど | 標準 | ★★☆ よく使う | 「なくなったわけじゃないんですけど」 |
| 〜くて / 〜なくて | 標準 | ★★☆ つなぎ | 「手につかなくて」 |
| 〜できて | 標準 | ★☆☆ ポジティブ転換 | 「ちゃんと集中できて」 |
| 〜のよな | 方言 | ★★☆ 1投稿2〜3回 | 「入ってこんのよな」 |
| 〜んよな | 方言 | ★★☆ 1投稿2〜3回 | 「大きいんよな」 |
| 〜んよね | 方言 | ★★☆ 1投稿2〜3回 | 「お願いしてみたんよね」 |
| 〜な気がする | 柔らかい断定 | ★☆☆ アクセント | 「〜だけじゃない気がする」 |
| 〜いなぁ / 〜かなぁ | 寄り添い | ★☆☆ 締め向き | 「届いたらいいなぁ」 |

### 2.2 語尾の使い方ルール

- **方言語尾は1投稿に2〜3回**。全部方言にはしない
- **標準語尾と方言語尾をMIX**して、自然なバランスを作る
- **同じ語尾を2回以上連続させない**
- 500字あるので**5〜6種類の語尾**を使い分ける

### 2.3 絶対に使わない語尾・表現

| NG表現 | 理由 |
|--------|------|
| 〜しましょう！ | 押し付け感がある |
| 〜してみてください！ | 指示っぽい |
| 〜してみて | 上から目線に聞こえる |
| いかがでしたか？ | AIっぽい |
| 〜ではないでしょうか | 評論家っぽい |
| 〜することが大切です | 教科書っぽい |
| 〜していきたいと思います | ブログの定型感 |

## 3. 文章構成

### 3.1 基本構成（6パート）

```
① 冒頭（今日の出来事 or 共感フック）（1〜2行）
  ↓
② 背景・きっかけ（2〜3行）
  ↓
③ 自分の感情・体験の描写（3〜5行）
  ↓
④ 転換（「でもね」「それがね」等）（1行）
  ↓
⑤ 小さな変化・気づき（2〜3行）
  ↓
⑥ 読んでる人への寄り添い（1〜2行）絵文字
  ↓
⑦ ハッシュタグ
```

### 3.2 各パートのポイント

**① 冒頭**: 「今日〜があった」で日記的に始めるか、「〜な人いる？」で共感から入る
**②③ 体験**: Threadsの見せ所。**Before（つらかった頃）を丁寧に描写**する。感情を具体的に、体の感覚で表現する（「心臓バクバク」「胸がすっと軽くなった」等）
**④ 転換**: 「でもね」「それがね」「でも最近」など柔らかい接続詞
**⑤ 変化**: 大げさにしない。「ちょっとだけ」「少しだけ」の温度感
**⑥ 寄り添い**: 「届いたらいいなぁ」「同じ人いるかなぁ」で余韻を残す

### 3.3 投稿パターン（4種類）

**パターン1: 共感日記型（メイン）**
```
[今日あったこと]（1〜2行）
[背景・前はどうだったか]（2〜3行）
[体験の描写]（2〜3行）絵文字
[転換]
[変化・気づき]（2〜3行）
[寄り添い]（1行）絵文字
ハッシュタグ
```

**パターン2: 振り返りストーリー型**
```
[過去のつらかった体験]（2〜3行）
[当時の気持ちを丁寧に描写]（3〜4行）
[転換: 今振り返ると]
[今だから思えること]（2〜3行）絵文字
[自分 or 同じ悩みの人への寄り添い]（1〜2行）
ハッシュタグ
```

**パターン3: 気づきシェア型**
```
[最近気づいたこと]（1〜2行）
[きっかけの出来事]（3〜4行）
[自分がどう感じたか]（2〜3行）
[問いかけ or 寄り添い]（1〜2行）絵文字
ハッシュタグ
```

**パターン4: note誘導型**
```
[読者の悩みに触れる]（2〜3行）
[自分も同じだった体験]（2〜3行）
[noteに書いたことをチラ見せ]（1〜2行）📝
ハッシュタグ
```

### 3.4 構成のNG

- 箇条書きで「方法3選！」のような情報まとめ型（Xやnoteの役割）
- 最初から最後まで辛い話だけ（最後に小さな光を入れる）
- 解決策を詳しく解説する（Threadsは「気づき」まで。詳細はnoteで）

## 4. トーン・言葉選び

### 4.1 全体のトーン

- **温度**: あたたかめ。Xよりほんの少し感情が出る
- **距離**: 友達。カフェで向かい合ってる感じ
- **テンション**: 穏やか。語りかけるペース
- **目線**: 同じ高さ。「私もそうだったんよね」の目線
- **時間感覚**: 「今日あったこと」「最近感じたこと」のリアルタイム感

### 4.2 よく使う言い回し

| シーン | 使う表現 | 使わない表現 |
|--------|---------|-------------|
| 冒頭 | 「今日〜があった」「最近〜なんですけど」 | 「皆さんに共有したいことが」 |
| 体験を語るとき | 「前までは〜だったのに」「ずっと〜で」 | 「〜という経験をしました」 |
| 感情を出すとき | 「自分でもびっくりしちゃって」「ほんとにしんどくて」 | 「非常に困難でした」 |
| 転換 | 「でもね」「それがね」「でも最近」 | 「しかし」「ところが」 |
| 変化を伝えるとき | 「ちょっとだけ楽になれた」「少しだけ変わった」 | 「劇的に改善した」「大きく変わった」 |
| 寄り添い | 「届いたらいいなぁ」「同じ人いるかなぁ」 | 「参考になれば幸いです」 |
| note誘導 | 「noteにまとめてみた📝」 | 「noteはこちら→」 |

### 4.3 感情描写のルール

- Xでは「しんどかった」→ Threadsでは「午前中ずっとソワソワして、なにも手につかなかった」のように**具体的に描写**
- 辛さも嬉しさも**体の感覚**で表現（「心臓バクバク」「胸がすっと軽くなった」等）
- Before → After の**差**を描くことで、小さな変化でも大きく感じてもらえる

### 4.4 「構造的な言語化」テクニック

- 「発言が怖い」→「センサー張りっぱなしで心がすり減ってる」
- 「疲れやすい」→「処理する情報量が多すぎて追いついてない」
- 「頑張れない」→「頑張りすぎて壊れかけてる」

## 5. 絵文字ルール

### 5.1 絵文字パレット（公式）

**メイン（よく使う）**:
| 絵文字 | 用途 |
|--------|------|
| 🫧 | ゆるさ、ふわっとした空気感 |
| 🌙 | 夜のつぶやき、落ち着いた雰囲気 |
| 🍃 | 自然体、心が軽くなった感 |
| 💭 | 考えてること、問いかけ |
| ✨ | ポジティブな変化、小さな光 |
| 🌿 | 穏やかさ、癒し |

**サブ（時々使う）**:
| 絵文字 | 用途 |
|--------|------|
| ☁️ | ふわっとした雰囲気、モヤモヤ |
| 🤍 | やさしさ、共感 |
| 🪄 | ちょっとした魔法、変化 |
| 🫶 | 寄り添い、応援 |
| 🧸 | やわらかさ、安心感 |
| 📝 | note誘導 |

### 5.2 使い方ルール

| ルール | 詳細 |
|--------|------|
| 使用量 | 1投稿に**2〜4個** |
| 配置 | 感情の転換点や締め付近。文中にばらまかない |
| 組み合わせ | 🍃✨（辛かったけど光が見えた）等は効果的 |
| NG絵文字 | 😢☺️🙄等の顔文字系、🔥💪🎉等のハイテンション系 |
| 世界観 | ayamiroomのような「ふわっとした空気感」を意識 |

## 8. AI生成時のチェックリスト

投稿案を生成した後、以下をチェック：

- [ ] 同じ語尾が2回以上連続していないか
- [ ] 方言語尾（のよな・んよな・んよね等）が1投稿に2〜3回に収まっているか
- [ ] 方言だけの投稿になっていないか（標準語尾とMIXされているか）
- [ ] 「〜しましょう」「〜してみて」「いかがでしたか」が含まれていないか
- [ ] 箇条書きだけの投稿になっていないか
- [ ] 顔文字系の絵文字（😢☺️🙄等）が使われていないか
- [ ] ハイテンション系の絵文字（🔥💪🎉等）が使われていないか
- [ ] 絵文字パレット外の絵文字が使われていないか
- [ ] 絵文字が5個以上使われていないか
- [ ] 500字以内に収まっているか
- [ ] 最後が押し付けや指示で終わっていないか
- [ ] note誘導が「noteはこちら→」的な直接誘導になっていないか
- [ ] 短文×改行のリズムが保たれているか
- [ ] 最初から最後まで辛い話だけになっていないか（最後に小さな光を入れる）
- [ ] 感情が具体的に描写されているか（体の感覚で表現されているか）
- [ ] Before → Afterの変化が「ちょっとだけ」の温度感か
