# Threads 文体・構成ガイドライン

**作成日：2026年2月28日**

---

## 1. Threadsの役割と基本方針

**役割**: 飾らない本音と個人的な体験談を語り、ターゲットとの間に強い共感と信頼関係を築く「深い共感とファン化の聖域」。

**基本方針**: 「やまひさんだから信頼できる」という熱量の高いファンを作る場所。有益性よりも**共感性**を最優先する。あなたのガイドラインを全面的に採用し、最も「あなたらしさ」が出る場所とする。

**参考モデル**: haru_hsp_92（共感＋本音）, chimacoro（強い共感＋GIVE）

---

## 2. 文体：カフェで向かい合う友達

### 2.1 基本トーン

- **丁寧さ**: 丁寧語ベースの優しい語りかけ口調。「〜なんですよね」「〜じゃないですか？」など、あなたのガイドラインにある語尾をメインに使う。
- **一人称**: 私
- **距離感**: 友達。カフェで「実はさ…」と話すような親密さ。
- **感情描写**: 「午前中ずっとソワソワして、なにも手につかなかった」のように、**体の感覚**で具体的に表現する。
- **方言**: 「〜んよな」は**1投稿に1〜2回**まで。スパイスとして使い、親近感を演出する。

### 2.2 言葉選び

- **共感を呼ぶ言葉**: 「わかります」「私もそうでした」「なんででしょうね」など、感情や疑問をそのまま言葉にする。
- **弱さの開示**: 弱さや葛藤も隠さず、等身大の自分で読者と同じ目線に立つ。

### 2.3 絵文字・ハッシュタグ

あなたのガイドラインを**完全採用**します。

- **絵文字パレット**: 🫧 🌙 🍃 💭 ✨ 🌿 をメインに、1投稿2〜4個。
- **ハッシュタグ**: 1投稿3〜5個。メインタグ（#HSP #HSPママ）＋テーマ別タグ（#キャリア #育児）＋具体シーンタグ（#仕組み化疲れ #デジタル断捨離）。

## 3. 構成テンプレート

あなたのガイドラインにある4つのストーリー型をメインに、Manus案のTips型をサブとして加えることで、発信に幅を持たせます。

### 3.1 A. 共感日記型（メイン）

**目的**: 日常の出来事を通して、共感と親近感を醸成する。

```
【1行目：共感フック】
（例）また周りの期待に応えすぎて、勝手に疲れちゃいました…。

【2行目以降：具体的なエピソード（体験談）】
良かれと思って先回りして仕事を進めたら、それが当たり前になって、
どんどんタスクが増えていくんです。
感謝もされず、なんだかモヤモヤするだけ。
HSPのこの気質、本当に悩ましいですよね…。

【中盤：気づき・学び】
でも最近、こう考えるようにしたんです。
「期待に応える」んじゃなくて、「自分の心を守る」ために仕事をするんだって。

【結び：読者への問いかけ or 励まし】
そのために私が手放した3つのこと、今度noteにまとめてみようと思います。
皆さんは、どんな時に心がすり減りますか？

#HSPあるある #仕組み化疲れ #デジタル断捨離
```

### 3.2 B. 振り返りストーリー型（メイン）

**目的**: 過去の失敗談や成功体験を語り、読者に勇気と希望を与える。

```
【フック】
3年前の私へ。

【過去の状況】
毎日、周りの顔色をうかがって、仕事でも家でも気を使いすぎて、夜にはもうヘトヘト。
「なんで私ばっかり…」
そんな風に、一人で泣いていたあなたへ。

【転換点】
でも大丈夫。3年後のあなたは、心をすり減らすことなく、
自分のペースで働き、家族と笑い合える毎日を手に入れています。

【現在の状況とメッセージ】
この投稿は、過去の私と同じように悩んでいるあなたが、
その未来を手に入れるための「小さなヒント」です。

#HSPの働き方 #キャリアチェンジ #過去の自分へ
```

### 3.3 C. 気づきシェア型（メイン）

**目的**: 日々の生活の中での小さな発見や学びを共有し、読者に新たな視点を提供する。

```
【フック】
最近、すごく腑に落ちたことがあって。

【気づきの内容】
「頑張る」って、アクセルを踏むことだけじゃないんですよね。
むしろ、HSPにとっては「ブレーキのかけ方を知ること」の方がずっと大事なんじゃないかなって。

【具体例】
例えば、
・疲れたら堂々と休む
・やりたくないことは断る
・完璧を目指さない

これって全部、上手なブレーキのかけ方。

【結び】
アクセル全開じゃなくても、ちゃんと前に進める。
むしろ、その方が遠くまで行ける気がするんです。

#HSPの生き方 #頑張らない #セルフケア
```

### 3.4 D. note誘導型（メイン）

**目的**: note記事への期待感を高め、クリックを促す。

```
【フック】
週末に公開するnote、少しだけお見せしますね。

【内容の一部を紹介】
テーマは「HSPのための『断るスキル』」。

・罪悪感なく断るための心理学
・波風を立てない魔法の言い回し3選
・断ることで、逆に信頼される理由

【ベネフィット】
この記事を読めば、あなたはもう人間関係で心をすり減らすことはありません。

【CTA】
公開は〇日(土)20時。Threadsのフォロワーさん限定で、記事には書けない裏話も少しだけ追記しようかな…。

#note #HSP #人間関係
```

### 3.5 E. Tips投稿型（サブ）

**目的**: サクッと読める有益な情報を提供し、「この人は有益だ」という認知を広げる。

```
【タイトル】
知ってるだけで心が軽くなるHSPさんのための思考法3選

【結論】
心がザワザワした時は、この3つを試してみて。

【Tips一覧】
①「べき」を「したい」に変換する
（例）メールは早く返すべき→返したい時に返す

② 自分の感情を実況中継する
（例）「あ、今、私焦ってるな」と客観視する

③ 小さな違和感を放置しない
（例）ランチの場所、本当はA店が良かったな…など

【まとめ】
自分の心の声を一番大切にしてあげてくださいね。

#HSP #自己肯定感 #ライフハック
```

## 4. AI生成時チェックリスト

あなたのガイドラインにある**20項目以上の詳細なチェックリスト**をそのまま活用します。これにより、AIが生成した投稿案も、あなたらしい繊細なニュアンスを保つことができます。

- [ ] 語尾は「〜なんですよね」「〜じゃないですか？」など、友達に語りかけるトーンになっているか？
- [ ] 方言語尾「〜んよな」は1〜2回に収まっているか？
- [ ] 感情が体の感覚で具体的に表現されているか？
- [ ] 5つの構成テンプレートのいずれかに沿っているか？
- [ ] 絵文字パレット内の絵文字が2〜4個使われているか？
- [ ] ハッシュタグが3〜5個、適切に設定されているか？
- [ ] 売り込み感がなく、あくまで「共有」のスタンスが保たれているか？
